01

辅助的核心定位与职责

辅助是团队的"隐形核心"，虽然不直接输出伤害，但却是整个团队运转的润滑剂。一个优秀的辅助能够通过视野、控制、保护来扭转战局，是决定团队上限的关键角色。

🛡️

保护核心输出

射手和法师是团队的主要输出点，辅助的首要任务是确保他们的安全

团战时，辅助的站位应当始终在己方脆皮和敌方刺客之间。当敌方刺客（如李白、阿轲、兰陵王）消失在视野中时，要格外警惕，随时准备用控制技能打断其突进。保护核心的关键在于预判，而不是等队友被攻击后才反应。

⚡

提供控制与开团

在合适的时机使用控制技能，为团队创造输出空间或发起团战

开团的时机选择至关重要。优先开敌方走位失误或落单的英雄；团战前等待敌方关键技能（如东皇太一大招、妲己技能）的CD；开团后要立即跟进后续控制，避免敌方逃脱。

👁️

布控视野

掌握敌情，为团队提供信息优势，预防敌方gank和埋伏

视野是团队的信息来源。在龙坑、中路草丛、河道等关键位置放置视野，能够让团队提前预判敌方动向。软辅优先用技能探草，硬辅可以直接进草丛脸探。

💪

承受伤害

在关键时刻挺身而出，为队友创造输出空间和逃生机会

辅助是团队中最"不怕死"的位置，但要死得有价值。判断团战局势，如果己方C位能够继续输出，辅助应当尽可能拖延时间吸引火力；如果C位已经被秒，则可以选择性撤退保存实力。

🗺️

游走支援

配合打野进行地图游走，帮助各路建立优势

游走不是盲目乱跑，而是有目的性地帮助队友建立优势。中路抢线后配合打野入侵野区、反蹲敌方游走英雄；或者保护射手快速推塔、建立对线优势。

📢

指挥与沟通

辅助是团队的大脑，需要及时传递信息和指挥决策

辅助因为不专注于操作，有更多精力观察全局。及时报点"敌方打野在中路"、"准备开龙"、"可以团了"等指令，能够让队友做出更好的决策。良好的沟通是团队配合的基础。

💡 核心思维

  • 辅助不是躺，是整个团队的润滑剂和守护者
  • 辅助的价值不在于击杀数，而在于创造多少有效击杀机会
  • 辅助的每一次牺牲，都应该是为了换取更大的团队优势
  • 视野做得好，比拿十个人头更有用

🎯 实战案例分析

场景：后期团战，敌方兰陵王消失。你方后羿正在输出，作为辅助张飞，你应该怎么做？

正确做法：立即靠近后羿，将一技能护盾保持在手边，大招随时准备喷开切入的兰陵王。即使兰陵王从后方切入，也要第一时间用大招将其吼开，保护后羿的输出环境。

错误做法：站在前排等待开团，没有注意到后方兰陵王的动向，导致后羿被秒杀后团战失败。

02

开局思路与开局技巧

开局的前30秒往往决定了前期的节奏走向。一个好的开局能够帮助团队建立前期优势，为中后期的团战奠定基础。辅助需要在开局时快速判断局势，做出最有利的选择。

1

帮打野看野区

30秒内确认对面打野开法

开局第一时间去敌方红/蓝buff区查看打野动向。站在buff草丛边缘观察而不进入，看到敌方打野位置后立即返回。如果敌方打野30秒时还未出现红蓝buff，说明他在另一个buff开。

2

中路抢线

配合法师快速清线

抢完视野后快速回到中路，帮助法师清理第一波兵线。抢线成功后可以：配合打野入侵野区、反蹲敌方游走、或者保护射手快速推塔。

→
3

选择支援方向

根据打野动向选择策略

如果打野准备入侵野区，辅助应该跟随；如果打野选择gank，辅助可以选择反蹲或配合；如果双方和平开局，优先帮助射手建立对线优势。

🎮 开局黄金法则

  • 不要盲目跟射手：开局阶段射手对线尚未开始，辅助应该先把视野和抢线做好
  • 关注小地图：开局30秒内敌方打野位置是核心信息，及时报点
  • 沟通是关键：开局前用语音或快捷消息告诉队友你的计划
  • 灵活应变：根据敌方阵容和己方打野动向随时调整策略
  • 保护自己：开局探视野时不要走太深，避免被敌方多人包夹

03

游走支援的时机判断

游走是辅助建立优势的核心手段，但盲目游走会导致射手被击杀、节奏崩盘。掌握游走的时机，是辅助进阶的关键技能。

✅ 适合游走的时机

  • ✓ 射手安全清线时（塔下收兵，无压力）
  • ✓ 己方兵线优势时（兵线推到敌方塔下）
  • ✓ 打野准备入侵/gank时
  • ✓ 敌方关键技能交完后（如敌方射手刚交闪）

判断标准：游走前看一眼小地图，确保三件事——射手位置安全、敌方打野不在附近、游走路线没有暴露风险。

❌ 不宜游走的时机

  • ✗ 射手被压线/推塔时（塔即将被拆）
  • ✗ 敌方打野消失（可能有埋伏）
  • ✗ 龙即将刷新（30秒内）
  • ✗ 己方血量不健康时（容易被反杀）

危险信号：如果不确定是否应该游走，那就留下来保护射手。保守的打法不会让你背锅。

🎯 游走路线选择

  • 中路河道游走：最常用路线，经过河道草丛时用技能探草
  • 己方野区绕行：如果敌方可能有埋伏，从己方野区绕行更安全
  • 配合打野入侵：入侵时保持与打野视野范围，不要单独行动
  • 反蹲策略：如果判断敌方可能来gank，可以在草丛反蹲等待机会

🎮 游走成功案例

场景：己方后羿和敌方鲁班七号对线，兵线在敌方塔下。你的大乔准备游走支援中路。

操作流程：

1. 确认后羿塔下收兵安全 → 2. 从河道快速移动到中路 → 3. 配合法师击杀敌方小乔 → 4. 利用大乔技能快速回线 → 5. 后羿已经推掉敌方一塔

结果：成功帮助中路建立优势，同时没有让射手掉塔，实现双赢。

⚠️ 游走注意事项

  • 游走后第一时间用信号通知射手，让他注意敌方游走
  • 游走时间不宜过长，超过10秒没有机会就立即回线
  • 软辅游走时要保持与队友的技能覆盖范围，以便随时支援
  • 如果游走失败导致射手死亡，这波游走就是亏的
04

团战站位与技能释放

团战是游戏中最激烈的时刻，辅助的站位和技能释放直接决定团战的走向。每一个控制技能、每一次保护，都可能成为团战胜负的关键。

📍 站位原则

  • 1. 始终处于己方脆皮和敌方刺客之间
  • 2. 与队友保持适当距离（技能覆盖范围）
  • 3. 不要站在最前方承受不明伤害
  • 4. 团战开始前抢占有利地形

地形利用：利用草丛、墙壁等地形，让敌方难以接近己方脆皮。团战时背靠墙壁可以防止被敌方突进绕后。

⚡ 技能释放时机

  • 控制技能：等敌方关键人物出现再交
  • 保护技能：己方脆皮被切时第一时间交
  • 增益技能：己方输出准备输出时交
  • 收割技能：敌方残血时交给打野收割

关键原则：控制技能不要交给敌方前排，留给敌方核心输出或突进的刺客。保护技能要在伤害到来前交，而不是等队友残血才交。

🎯

硬辅团战策略

开团型：寻找敌方走位失误，用控制技能开团，团战开始后留在前排吸收伤害，为队友创造输出环境。

保护型：紧跟己方脆皮，敌方刺客切入时立即用控制技能打断，保护后排输出。

🔮

软辅团战策略

增益型：保持与核心输出在技能范围内，持续提供增益效果。利用手长优势在安全位置输出和控制。

拉扯型：不要与敌方近身，利用技能拉扯阵型，边打边退，寻找输出空间。

🎮 团战技能释放时机详解

场景：团战即将爆发，敌方有关羽、娜可露露、妲己。己方有后羿、小乔、张飞。

分析：

• 关羽是最大威胁，会绕后推人，需要提前预留控制

• 娜可露露会直接突进后排，需要重点保护

• 妲己控制+爆发一套秒人，要等妲己先手后再交保命技能

正确操作：等妲己先手控住后羿后，张飞立即开大喷开娜可露露，同时给后羿套盾。一技能等待关羽切入时打断其冲锋。

05

保护后排的优先级

团战中辅助不可能同时保护所有人，需要根据英雄特性和团战局势判断保护优先级。正确的保护优先级能够让团队输出最大化。

①

射手

团战持续输出的保障

保护要点：射手通常需要4-5秒持续输出来击杀前排，如果被切第一时间交保护技能
核心输出
②

法师

团控 + 高额AOE伤害

保护要点：法师通常有范围伤害技能，存活下来可以持续输出AOE，注意敌方刺客
爆发输出
③

打野

需要空间完成收割

保护要点：打野通常有收割能力，但前期较脆，如果打野有保命技能可适当放养
切后担当
④

战士/坦克

有续航和自保能力

保护要点：战士和坦克本身有一定坦度，除非被集火否则优先保脆皮
前排

⚡ 特殊情况处理

  • 当我方射手被秒杀时：立即转移保护正在输出的队友（可能是法师或打野）
  • 当我方射手有复活甲时：优先保护其他脆皮，给射手站起来反打的机会
  • 当我方射手是大神时：即使牺牲自己也要保住核心输出
  • 当敌方有真实伤害时：坦克的保命能力下降，需要更多保护

⚠️ 保护优先级判断陷阱

  • 不要为了保护残血射手而交掉所有保护技能，导致后续无人可保
  • 如果己方射手已经被秒且无法救回，不要强行保护浪费技能
  • 保护时注意自己的位置，不要为了救人把自己也送掉
  • 团战中保护优先级是动态变化的，需要实时判断
06

视野布控的艺术

视野是团队的信息来源，好的视野布控能够让团队提前预判敌方动向，做出最优决策。辅助是团队的主要视野提供者，需要养成良好的做视野习惯。

👁️ 关键视野点

  • • 龙坑周围的草丛：判断敌方开龙或准备抢龙
  • • 河道草丛：敌方游走支援的主要路线
  • • 己方野区入口：预防敌方入侵和gank
  • • 敌方野区buff草丛：判断敌方打野动向和节奏
  • • 中路两侧草丛：敌方中路游走的必经之路

视野优先级：龙坑视野 > 河道视野 > 野区视野。关键资源点附近的视野最为重要。

⏰ 布眼时机

  • • 每次回家前：确保关键草丛有眼再走
  • • 团战前：提前布置进攻视野
  • • 龙刷新前30秒：开始布控龙坑视野
  • • 推塔时：在敌方野区做进攻视野
  • • 队友单带时：在单带路线做保护视野

习惯养成：每次死亡后复活，先做视野再支援。养成习惯比临时抱佛脚更有效。

🎮 探草技巧

  • 软辅：利用远程技能探草（如孙膑一技能、小明链子）
  • 硬辅：可以直接脸探草丛，利用护盾或霸体承受伤害
  • 瑶：附身时可以用一技能远程探草
  • 大乔：可以在草丛放技能沉默范围探测
  • 鬼谷子：利用一技能加速进草探测
🗺️

不同阶段的视野策略

前期（0-4分钟）：重点关注敌方打野位置和中路河道，防止gank。

中期（4-10分钟）：重点关注龙坑视野和敌方野区，判断敌方发育情况。

后期（10分钟+）：重点关注龙坑和敌方高地附近视野，获取团战优势信息。

💀

视野被反蹲怎么办

预防措施：做视野时不要一个人，从己方野区绕行，让队友知道你的位置。

被抓后的应对：立即向队友发送"敌人消失"信号，让他们及时撤退或反打。

注意：做视野牺牲比被突然抓单损失小，必要时可以换掉敌方一个核心。

07

不同辅助的玩法差异

辅助根据技能特性可以分为多种类型，每种类型有不同的打法思路。了解自己使用的辅助类型，能够让你更好地发挥英雄特点。

硬辅

张飞、牛魔、盾山、廉颇

开团 + 保护

特点：拥有强大的控制技能和坦度，是团队的前排担当。

打法：开团时寻找敌方走位失误，保护时紧跟脆皮。硬辅可以主动脸探草丛，为团队开视野。

代表英雄技巧：

• 张飞：大招尽量喷到更多人，一技能护盾看准时机给

• 牛魔：大招蓄力时观察地形，可以配合闪现使用

• 盾山：一技能举盾挡飞行物，二闪开团

软辅

大乔、孙膑、瑶、桑启

增益 + 拉扯

特点：拥有强大的增益效果和保护能力，擅长拉扯作战。

打法：保持与队友的技能范围，利用增益效果帮助队友建立优势。不要与敌方近身作战。

代表英雄技巧：

• 大乔：电梯流打法，回城后立即返回战场

• 孙膑：二技能抬血加速，大招沉默打断敌方技能

• 瑶：附身时注意盾的厚度，及时下来挡技能

法辅

姜子牙、太乙真人、西施

速刷 + 爆发

特点：拥有强大的增益效果和范围伤害，团队经济来源。

打法：利用被动技能帮助队友快速发育，团战中提供范围伤害和控制。

代表英雄技巧：

• 姜子牙：被动加速全队发育，大闪收割

• 太乙：大招复活看准时机，一技能蓄力晕人

• 西施：一技能拉人，二技能范围伤害

坦辅

猪八戒、苏烈、钟无艳

承伤 + 控制

特点：拥有高额的生命值和护盾，擅长吸收伤害和控制。

打法：站在前排吸收伤害，为队友创造输出空间。技能有控制效果，可以用来开团或保护。

代表英雄技巧：

• 猪八戒：残血时利用技能回血，换血能力强

• 苏烈：大招蓄力接闪现，二技能推人

• 钟无艳：石化控制，伤害高可以一套秒脆皮

📋 辅助选人建议

  • 己方脆皮多：选择硬辅或坦辅，提供前排坦度和保护
  • 己方战士多：选择软辅，提供增益和拉扯能力
  • 敌方突进多：选择控制型辅助，重点保护后排
  • 己方需要发育：选择法辅，利用被动加速团队经济
  • 敌方有姜子牙：己方一定要选法辅对冲被动
08

常见错误与改正方法

了解常见的辅助错误，能够帮助你避免踩坑，快速提升游戏水平。以下是辅助玩家最容易犯的错误，以及正确的改正方法。

一直跟着射手不游走

开局就死跟射手，忽略其他路支援和视野

→

学会判断游走时机

射手安全清线时立即游走支援中路或配合打野，解放射手的同时帮助其他路建立优势

盲目开团不懂保护

只顾着开团，忽略了后排保护

→

根据局势选择开团或保护

己方脆皮多时以保护为主，己方战士多时可以开团。团战中根据敌方突进情况灵活切换角色

视野布控不到位

不做视野或做的视野位置不好

→

养成习惯性放眼的习惯

每次回家前确保关键草丛有眼，龙刷新前30秒开始布控视野。养成主动做视野的习惯

技能乱交不保留

团战中技能随便交，关键时刻没技能

→

关键技能留给关键目标

控制技能留给敌方核心输出或突进的刺客，保护技能等队友被切时再交，不要提前浪费

不关注小地图

只盯着自己的位置，不知道全局情况

→

养成每3秒看一次的习惯

每3秒扫一眼小地图，了解敌方位置和动向。辅助不专注操作，有更多精力观察全局

不会主动探草

怕死不敢进草丛，队友被蹲也不管

→

用技能或身体探草

软辅用技能探草，硬辅可以直接脸探草丛。牺牲自己换队友安全是值得的

不会报点和沟通

看到敌人也不说话，默默单机

→

主动报点，让队友知道信息

看到敌方打野位置立即报点"打野在中"，准备开龙时发信号"准备开龙"。良好的沟通是胜利的基础

辅助玩家的成长路线

辅助意识的提升是一个循序渐进的过程。从基础保护到高级指挥，每个阶段都有不同的学习重点。以下是辅助玩家的四个成长阶段：

🌱

第一阶段：青铜~白银 - 基础保护

学会基本的保护后排，不乱跑，跟紧队友

跟紧射手 不送人头 放技能保护
🚀

第二阶段：黄金~铂金 - 视野与游走

学会做视野，了解游走时机，开始主动支援

主动做视野 判断游走时机 配合队友
⚔️

第三阶段：钻石~星耀 - 开团与节奏

学会开团时机，掌握团队节奏，指挥决策

开团时机 团队节奏 指挥决策
👑

第四阶段：王者~荣耀 - 全局意识

了解每个英雄特性，预判敌方行动，成为团队大脑

英雄理解 预判能力 全局指挥

📈 快速提升建议

  • 多看小地图：这是辅助最重要的习惯，养成每3秒扫一眼的习惯
  • 学会报点：看到敌方位置立即发信号，让队友知道信息
  • 复盘思考：每局结束后思考自己哪里做得不好，下次如何改进
  • 学习对线：了解射手和法师的对线技巧，才能更好地保护他们
  • 观看教学：多看高水平辅助玩家的直播或视频，学习他们的意识

🎯 辅助进阶核心：全局思维

核心理念：辅助是团队的眼睛和大脑，不是操作的执行者。

高水平辅助与低水平辅助的最大区别在于：

• 低水平辅助：等队友被打了才反应过来，看到敌人不知道该干嘛

• 高水平辅助：提前预判敌方动向，主动创造机会，让队友在安全环境下输出

关键转变：从"保护队友"到"让队友安全"——不是等队友被打了才保护，而是让队友根本不会被攻击。

∞

辅助意识核心总结

记住这五大核心原则，让你从普通辅助晋升为团队核心：

🛡️ 保护后排 > 开团冲动
👁️ 视野布控 > 站桩输出
🗺️ 游走支援 > 赖线发育
📢 沟通报点 > 默默单机
🧠 观察小地图 > 只看眼前
⏱️ 技能时机 > 乱交技能

🌟 最终寄语

辅助是团队中最容易被低估的位置，但也是最能给团队带来胜利的位置。

一个优秀的辅助，能够让队友在安全的环境下全力输出；
一个顶级的辅助，能够提前预判一切，让敌方根本没有进攻的机会。

愿你在辅助的道路上越走越远，成为队友最信赖的守护者！
