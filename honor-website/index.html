01

辅助到底该干嘛

说实话，辅助这个位置挺冤的。很多人觉得辅助就是混，但其实一局打下来，辅助要做的事情最多。你要保人、开团、做视野、还要指挥，简直像个老妈子一样操心全队。

🛡️

保住你家C位

射手和法师是打输出的，你死了没事，他们死了团战基本就凉了

团战的时候，你就站在自家脆皮和对面刺客中间。看到对面李白、阿轲、兰陵王不见了，赶紧往后靠，别傻乎乎站在前面。我玩张飞的时候，大招永远留着给对面切后排的，一技能护盾也捏在手里，刺客一露头就给他喷走。

⚡

开团或者反打

找到对面走位失误的人，或者等对面冲上来的时候反打

开团不是看到人就上，要看时机。对面有人落单了、或者关键技能交了（比如东皇大招用了），这时候再上。开团之后别光顾着追人，看看自家后排有没有被切。

👁️

做视野

让队友知道敌人在哪，这比拿人头重要多了

龙坑、河道、草丛这些关键地方要常去看看。软辅用技能探草，硬辅直接进去看。别怕死，你死了换队友不被蹲，这波不亏。我最烦那种躲在后面不敢探草的辅助，队友被蹲了还怪别人没意识。

💪

抗伤害

该你死的时候别怂，但别白给

辅助是团队里最"不值钱"的，但该死的时候得死得有价值。你家射手还在输出，你就顶上去拖延时间；如果射手已经没了，那你也别送了，赶紧撤。我见过太多辅助，射手都死光了还冲上去1打5，图啥呢？

🗺️

跟着打野搞事情

配合打野入侵、抓人，帮队友建立优势

游走不是瞎跑，是有目的的。中路抢完线看看能不能配合打野入侵对面野区，或者去反蹲对面游走的辅助。别一直赖在下路，射手没压力的时候你就去别的地方搞事情。

📢

报点沟通

辅助是最闲的位置，多看看地图给队友报信息

你不需要像打野那样专注刷野，也不需要像射手那样专注对线，所以你有更多精力看地图。看到对面打野在中路，赶紧发个信号；准备开龙了，提前跟队友说。沟通好了，胜率能高一大截。

💡 别再说辅助是混子了

  • 辅助不是躺，是全队最操心的那个
  • 辅助的价值不在击杀数，而在创造了多少机会
  • 辅助每次死，都要死得有价值
  • 视野做好了，比拿十个人头还管用

🎯 说个真实的例子

场景：后期团战，对面兰陵王不见了。你家后羿正在疯狂输出，你玩张飞，这时候你该怎么办？

正确做法：赶紧贴到后羿旁边，一技能护盾捏手里，大招随时准备喷。兰陵王从后面切进来的时候，第一时间把他吼走，保住后羿的输出空间。这波你死了没事，后羿活着就能赢。

错误做法：站在前面等开团，完全没注意后面，兰陵王一套把后羿秒了，团战直接崩盘。然后队友还打字"辅助在干嘛？"，你说冤不冤。

02

开局咋玩

开局前30秒特别关键，很多辅助一上来就往下路跑，其实大错特错。开局你要先搞清楚对面打野在哪开，这样全队都知道该怎么打。

1

先看对面打野

30秒内搞清楚对面红开还是蓝开

开局直接往对面红buff或者蓝buff那边走，站在草丛边缘看一眼就行，别深入。看到对面打野在哪边开，马上回来。如果30秒了红蓝buff都没人，那说明他在另一边开。

→
2

帮中路抢线

配合法师快速清第一波兵

看完打野位置赶紧回中路，帮法师清第一波兵线。抢完线之后可以配合打野入侵、反蹲，或者去帮射手推塔，选择很多。

→
3

决定去哪

看打野动向选择支援方向

打野要入侵野区，你就跟着去；打野要抓人，你就去反蹲或者配合；如果两边都和平开局，那就去帮射手建立优势。

🎮 开局记住这几条

  • 别一上来就跟射手：开局射手还没开始对线呢，你先把视野和抢线做好
  • 多看小地图：开局30秒内对面打野的位置是最重要的信息，赶紧报给队友
  • 跟队友说一声：开局前用语音或者快捷消息告诉队友你要干嘛
  • 灵活点：根据对面阵容和自家打野的动向随时调整
  • 别走太深：探视野的时候别一个人冲到对面人堆里，容易被包

03

什么时候该游走

游走是辅助建立优势的核心手段，但瞎游走会导致射手被单杀，节奏直接崩盘。什么时候该走、什么时候该留，这是区分好辅助和坑辅助的关键。

✅ 可以游走的时候

  • ✓ 射手在塔下安全清线，没压力
  • ✓ 兵线推到对面塔下了
  • ✓ 打野准备入侵或者抓人
  • ✓ 对面关键技能交了（比如射手闪现用了）

判断标准：游走前看一眼小地图，确认三件事——射手位置安全、对面打野不在附近、游走路线不会被蹲。

❌ 别瞎跑的时候

  • ✗ 射手被压线，塔都快没了
  • ✗ 对面打野不见了（可能被蹲）
  • ✗ 龙快刷新了（30秒内）
  • ✗ 自己血量不健康（容易被反杀）

危险信号：不确定该不该游走的时候，就老老实实待着保护射手。保守打法至少不会背锅。

🎯 游走路线怎么选

  • 中路河道：最常用的路线，经过草丛的时候用技能探一下
  • 自家野区绕：如果感觉对面可能在蹲，就从自家野区走，安全一点
  • 配合打野入侵：入侵的时候跟紧打野，别一个人乱跑
  • 反蹲：判断对面可能来抓人的时候，提前在草丛里等着

🎮 说个成功的游走

场景：你家后羿和对面鲁班对线，兵线推到对面塔下了。你玩大乔，想去支援中路。

操作流程：

1. 确认后羿塔下收兵安全 → 2. 从河道快速移动到中路 → 3. 配合法师击杀对面小乔 → 4. 用大乔技能快速回线 → 5. 后羿已经推掉对面一塔

结果：中路优势也建立了，射手塔也没掉，双赢。这种游走才是有用的游走。

⚠️ 游走别犯这些错

  • 游走了记得给射手发个信号，让他注意对面也来游走了
  • 游走时间别太长，超过10秒没机会就赶紧回去
  • 软辅游走的时候保持在队友技能范围内，方便随时支援
  • 如果因为你游走导致射手死了，这波就是亏的，别找借口
04

团战怎么站位和放技能

团战是游戏最刺激的时候，辅助的站位和技能释放直接决定团战输赢。一个控制技能放好了，可能就直接翻盘；放错了，队友全得跟着陪葬。

📍 站位怎么站

  • 1. 始终站在自家脆皮和对面刺客中间
  • 2. 和队友保持适当距离，技能能覆盖到就行
  • 3. 别傻乎乎站在最前面吃不明伤害
  • 4. 团战开始前先占好有利地形

地形利用：背靠墙壁或者躲在草丛旁边，这样对面刺客很难绕后。团战时站位比操作更重要。

⚡ 技能什么时候放

  • 控制技能：等对面关键人物出现再交
  • 保护技能：自家脆皮被切的时候第一时间交
  • 增益技能：队友准备输出的时候交
  • 收割技能：留给打野或者射手收人头

关键原则：控制技能别随便交给对面坦克，留给对面核心输出或者刺客。保护技能要在伤害来之前交，等队友残血了再交就晚了。

🎯

硬辅团战怎么打

开团型：找对面走位失误的人，控制技能开团。开团后留在前排吸收伤害，给队友创造输出环境。

保护型：紧跟自家脆皮，对面刺客切入的时候立即打断，保护后排输出。

🔮

软辅团战怎么打

增益型：保持在核心输出的技能范围内，持续给增益效果。利用手长优势在安全位置输出和控制。

拉扯型：别和对面近身，利用技能拉扯阵型，边打边退，寻找输出空间。

🎮 团战技能释放详解

场景：团战要爆发了，对面有关羽、娜可露露、妲己。你家有后羿、小乔、张飞。

分析：

• 关羽会绕后推人，要提前预留控制技能

• 娜可露露会直接突进后排，需要重点保护

• 妲己控制加爆发一套秒人，要等她先手后再交保命技能

正确操作：等妲己先手控住后羿后，张飞立即开大喷开娜可露露，同时给后羿套盾。一技能留着等关羽切入的时候打断他的冲锋。

05

团战先保谁

团战里你不可能同时保护所有人，得根据情况判断先保谁。保对了人，团战就能赢；保错了人，全队一起gg。

1

射手

团战持续输出的保障

保射手要点：射手通常需要4-5秒持续输出来击杀前排，如果被切第一时间交保护技能
核心输出
2

法师

团控 + 高额AOE伤害

保法师要点：法师通常有范围伤害技能，存活下来可以持续输出AOE，注意敌方刺客
爆发输出
3

打野

需要空间完成收割

保打野要点：打野通常有收割能力，但前期较脆，如果打野有保命技能可适当放养
切后担当
4

战士/坦克

有续航和自保能力

保战士要点：战士和坦克本身有一定坦度，除非被集火否则优先保脆皮
前排

⚡ 特殊情况怎么处理

  • 射手被秒了：赶紧转移保护正在输出的队友，可能是法师或者打野
  • 射手有复活甲：优先保护其他脆皮，给射手站起来反打的机会
  • 射手是大神：即使牺牲自己也要保住核心输出，大神能带你赢
  • 对面有真伤：坦克的保命能力下降，需要更多保护

⚠️ 保护优先级别踩这些坑

  • 别为了保护残血射手交掉所有技能，后面没人可保了
  • 如果射手已经被秒且救不回来，别强行保护浪费技能
  • 保护的时候注意自己的位置，别为了救人把自己也送了
  • 团战中保护优先级是动态变化的，要实时判断
06

视野怎么做

视野是团队的信息来源，视野做好了，队友就知道该干嘛；视野做不好，全队都像瞎子一样被对面牵着鼻子走。辅助是主要做视野的人，这个习惯一定要养成。

👁️ 关键视野点

  • • 龙坑周围的草丛：判断对面开龙或者准备抢龙
  • • 河道草丛：对面游走支援的主要路线
  • • 自家野区入口：预防对面入侵和gank
  • • 对面野区buff草丛：判断对面打野动向
  • • 中路两侧草丛：对面中路游走的必经之路

视野优先级：龙坑视野 > 河道视野 > 野区视野。关键资源点附近的视野最重要。

⏰ 什么时候做视野

  • • 每次回家前：确保关键草丛有眼再走
  • • 团战前：提前布置进攻视野
  • • 龙刷新前30秒：开始布控龙坑视野
  • • 推塔时：在对面野区做进攻视野
  • • 队友单带时：在单带路线做保护视野

习惯养成：每次死亡后复活，先做视野再支援。养成习惯比临时抱佛脚更有效。

🎮 探草技巧

  • 软辅：利用远程技能探草（孙膑一技能、小明链子）
  • 硬辅：可以直接脸探草丛，利用护盾或霸体承受伤害
  • 瑶：附身时可以用一技能远程探草
  • 大乔：可以在草丛放技能沉默范围探测
  • 鬼谷子：利用一技能加速进草探测
🗺️

不同时间段视野重点

前期（0-4分钟）：重点关注对面打野位置和中路河道，防止gank。

中期（4-10分钟）：重点关注龙坑视野和对面野区，判断对面发育情况。

后期（10分钟+）：重点关注龙坑和对面高地附近视野，获取团战优势信息。

💀

做视野被蹲了怎么办

预防：做视野别一个人去，从自家野区绕行，让队友知道你的位置。

被抓后：立即给队友发"敌人消失"信号，让他们及时撤退或者反打。

注意：做视野牺牲比被突然抓单损失小，必要时可以换掉对面一个核心。

07

不同辅助怎么玩

辅助根据技能特性可以分为好几种类型，每种类型打法都不一样。了解自己玩的辅助是什么类型，才能发挥出最大作用。

硬辅

张飞、牛魔、盾山、廉颇

开团 + 保护

特点：肉、控制强，是团队的前排担当。

打法：开团找对面走位失误的人，保护时紧跟脆皮。硬辅可以主动脸探草丛，为团队开视野。

英雄小技巧：

• 张飞：大招尽量喷到更多人，一技能护盾看准时机给

• 牛魔：大招蓄力时观察地形，可以配合闪现使用

• 盾山：一技能举盾挡飞行物，二闪开团

软辅

大乔、孙膑、瑶、桑启

增益 + 拉扯

特点：增益效果强，擅长拉扯作战。

打法：保持与队友的技能范围，利用增益效果帮助队友建立优势。别跟敌方近身。

英雄小技巧：

• 大乔：电梯流打法，回城后立即返回战场

• 孙膑：二技能抬血加速，大招沉默打断敌方技能

• 瑶：附身时注意盾的厚度，及时下来挡技能

法辅

姜子牙、太乙真人、西施

速刷 + 爆发

特点：增益效果和范围伤害，帮助团队快速发育。

打法：利用被动技能帮助队友快速发育，团战中提供范围伤害和控制。

英雄小技巧：

• 姜子牙：被动加速全队发育，大闪收割

• 太乙：大招复活看准时机，一技能蓄力晕人

• 西施：一技能拉人，二技能范围伤害

坦辅

猪八戒、苏烈、钟无艳

承伤 + 控制

特点：血厚、护盾多，擅长吸收伤害和控制。

打法：站在前排吸收伤害，为队友创造输出空间。技能有控制效果，可以用来开团或保护。

英雄小技巧：

• 猪八戒：残血时利用技能回血，换血能力强

• 苏烈：大招蓄力接闪现，二技能推人

• 钟无艳：石化控制，伤害高可以一套秒脆皮

📋 辅助选人建议

  • 自家脆皮多：选硬辅或者坦辅，提供前排坦度和保护
  • 自家战士多：选软辅，提供增益和拉扯能力
  • 对面突进多：选控制型辅助，重点保护后排
  • 自家需要发育：选法辅，利用被动加速团队经济
  • 对面有姜子牙：己方一定要选法辅对冲被动
08

辅助常见的坑

了解常见的辅助错误，能够帮助你避免踩坑，快速提升游戏水平。以下是辅助玩家最容易犯的错误，以及正确的改正方法。

一直跟着射手不游走

开局就死跟射手，忽略其他路支援和视野

→

学会判断游走时机

射手安全清线时立即游走支援中路或配合打野，解放射手的同时帮助其他路建立优势

盲目开团不懂保护

只顾着开团，忽略了后排保护

→

根据局势选择开团或保护

己方脆皮多时以保护为主，己方战士多时可以开团。团战中根据敌方突进情况灵活切换角色

视野布控不到位

不做视野或做的视野位置不好

→

养成习惯性放眼的习惯

每次回家前确保关键草丛有眼，龙刷新前30秒开始布控视野。养成主动做视野的习惯

技能乱交不保留

团战中技能随便交，关键时刻没技能

→

关键技能留给关键目标

控制技能留给敌方核心输出或突进的刺客，保护技能等队友被切时再交，不要提前浪费

不关注小地图

只盯着自己的位置，不知道全局情况

→

养成每3秒看一次的习惯

每3秒扫一眼小地图，了解敌方位置和动向。辅助不专注操作，有更多精力观察全局

不会主动探草

怕死不敢进草丛，队友被蹲也不管

→

用技能或身体探草

软辅用技能探草，硬辅可以直接脸探草丛。牺牲自己换队友安全是值得的

不会报点和沟通

看到敌人也不说话，默默单机

→

主动报点，让队友知道信息

看到敌方打野位置立即报点"打野在中"，准备开龙时发信号"准备开龙"。良好的沟通是胜利的基础

09

辅助怎么变强

辅助意识的提升是一个循序渐进的过程。从基础保护到高级指挥，每个阶段都有不同的学习重点。以下是辅助玩家的四个成长阶段：

🌱

青铜~白银 - 基础保护

学会基本的保护后排，不乱跑，跟紧队友

跟紧射手 不送人头 放技能保护
🚀

黄金~铂金 - 视野与游走

学会做视野，了解游走时机，开始主动支援

主动做视野 判断游走时机 配合队友
⚔️

钻石~星耀 - 开团与节奏

学会开团时机，掌握团队节奏，指挥决策

开团时机 团队节奏 指挥决策
👑

王者~荣耀 - 全局意识

了解每个英雄特性，预判敌方行动，成为团队大脑

英雄理解 预判能力 全局指挥

📈 快速提升建议

  • 多看小地图：这是辅助最重要的习惯，养成每3秒扫一眼的习惯
  • 学会报点：看到敌方位置立即发信号，让队友知道信息
  • 复盘思考：每局结束后思考自己哪里做得不好，下次如何改进
  • 学习对线：了解射手和法师的对线技巧，才能更好地保护他们
  • 观看教学：多看高水平辅助玩家的直播或视频，学习他们的意识

🎯 辅助进阶核心：全局思维

核心理念：辅助是团队的眼睛和大脑，不是操作的执行者。

高水平辅助与低水平辅助的最大区别在于：

• 低水平辅助：等队友被打了才反应过来，看到敌人不知道该干嘛

• 高水平辅助：提前预判敌方动向，主动创造机会，让队友在安全环境下输出

关键转变：从"保护队友"到"让队友安全"——不是等队友被打了才保护，而是让队友根本不会被攻击。

∞

辅助意识核心总结

记住这几条，让你从普通辅助变成团队核心：

🛡️ 保护后排 > 开团冲动
👁️ 视野布控 > 站桩输出
🗺️ 游走支援 > 赖线发育
📢 沟通报点 > 默默单机
🧠 观察小地图 > 只看眼前
⏱️ 技能时机 > 乱交技能

🌟 最后说两句

辅助是团队里最被低估的位置，但也是最能带来胜利的位置。

一个优秀的辅助，能让队友在安全的环境下全力输出；
一个顶级的辅助，能提前预判一切，让对面根本没有进攻的机会。

希望你在辅助的路上越走越远，成为队友最信赖的守护者！
